Catatan: Tool ini hanya mengonversi link ke format GitHub Pages, namun tidak dapat memastikan apakah halaman GitHub Pages tersebut sudah aktif atau tidak.

Tutorial 1: Cara Membuat GitHub Pages

  1. Buka repository GitHub yang ingin Anda gunakan untuk GitHub Pages.
  2. Pilih tab Settings di repository tersebut.
  3. Scroll ke bawah hingga menemukan section GitHub Pages.
  4. Setel Source menjadi main branch atau branch lainnya sesuai kebutuhan.
  5. Setelah diaktifkan, GitHub Pages Anda akan tersedia di link dengan format: https://username.github.io/repository.

Tutorial 2: Cara Convert GitHub Repository ke GitHub Pages

  1. Salin link GitHub repository yang ingin di-convert. Sebagai contoh: https://github.com/username/repository. Pastikan link tersebut valid dan mengarah ke repository yang benar.
  2. Buka halaman ini dan masukkan link repository GitHub yang telah Anda salin ke dalam kotak input di atas.
  3. Setelah memasukkan link, klik tombol Convert. Jika format link sudah benar, link GitHub Pages Anda akan dihasilkan secara otomatis di bawah kotak input.
  4. Salin atau klik link tersebut untuk membuka GitHub Pages Anda di tab baru. Pastikan GitHub Pages sudah diaktifkan di pengaturan repository Anda.

Tutorial 3: Cara Convert GitHub Pages ke GitHub Repository

Sebelum melanjutkan, pastikan Anda telah mencoba tools kedua untuk membalik konversi dari GitHub Pages ke GitHub repository.

  1. Jika Anda memiliki link GitHub Pages (contoh: https://username.github.io/repository), dan ingin mengembalikan link tersebut ke repository asli, buka tools kedua.
  2. Masukkan link GitHub Pages tersebut ke tools kedua.
  3. Klik tombol Convert untuk mendapatkan kembali link GitHub repository asli.
  4. Link GitHub repository akan muncul di bawah, dan Anda dapat menyalin atau mengunjungi repository tersebut.
